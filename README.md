# Smart-Watch
A custom low-power stm32-based wearable smart watch

## Features
- 6-Axis IMU Sensor for motion detections and pedometer.
- Temperature Sensor
- OLED Display
- Powered by 3.7V battery or USB
- 2 programmable switches
